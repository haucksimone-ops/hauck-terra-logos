Como Hospedar no seu GitHub Pages

Publique este gerador online e de graça em menos de 2 minutos

1

Renomeie o arquivo

Para o GitHub Pages reconhecer como a página inicial, mude o nome do arquivo baixado:

gerador_logotipos_hauck_terra.html ➜ index.html
2

Crie um novo Repositório

Vá no seu GitHub (github.com/new), crie um repositório chamado por exemplo hauck-terra-logos e marque como Público.

3

Faça o Upload do seu index.html

Na página do seu repositório criado:

  • Clique em "uploading an existing file".
  • Arraste e solte o arquivo index.html.
  • Arraste também a imagem cópia de neural_telemetry_orb_1780490653834.jpg para a mesma pasta raiz se quiser que o cérebro venha carregado por padrão!
  • Clique em "Commit changes" para salvar.
4

Ative o GitHub Pages

No topo do repositório no GitHub:

  1. Vá na aba Settings (Configurações).
  2. No menu lateral esquerdo, clique em Pages.
  3. Em Build and deployment ➜ Source, selecione Deploy from a branch.
  4. Em Branch, altere de None para main (ou master) e a pasta para / (root).
  5. Clique no botão Save.
Prontinho! 🎉

Em cerca de 1 minuto seu link estará ativo em:

https://seu-usuario.github.io/seu-repositorio/
